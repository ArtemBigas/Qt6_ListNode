# Qt6_ListNode
Тестовая работа для компании Effective Mobile  
Программа создана в Qt6, используя компилятор Visual Studio 2022  
Для удобства использования создан лаконичный интерфейс.    
Вместо имени файла,можете вставить полный адрес.  
Поскольку в задании нет указания как должна быть реализована Десериализация, её результат было решено выводить в самой программе.   
В качестве презентации, выложена релизная версия программы для ОС Windows 10 и Linux(Debian)
